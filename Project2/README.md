# IJC PROJECT 2
My solution to second IJC project

## Compilation:

ISO C11 for *.c files

```bash
gcc -std=c11 -pedantic -Wall -Wextra {file}.c
```

and ISO C++17 for *.cc files

```bash
g++ -std=c++17 -pedantic -Wall {file}.cc
```
